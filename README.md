# TopicViewerの使い方

## 動作環境
`node.js v6.9.1`で動作を確認した。


## メッセージの閲覧
あるトピックに貯めてあるメッセージを閲覧する場合、下記のコマンドを入力する。

```
npm run viewtopic -- <TopicName>
```

例えば、`test`トピックの場合は下記のようになる。

```
npm run viewtopic -- test
```

## 終了方法
`Ctrl+c`で終了できる。
